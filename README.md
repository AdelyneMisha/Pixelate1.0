# Pixelate1.0

Pixelate is a website aimed to generate patterns for various art involving pixels like cross stitch, diamond painting, tapestry designs etc.
It works by taking a user input image and then pixelating it.

This application uses Canvas API for pixelating images.

## This is a work in progress project. At present images are simply pixelated. This cannot be used to build art projects as a single pixel image can have thousands of colours in its pixels making it difficult to replicate in art in real life. Therefore more improvement is needed for this project to be functional.
